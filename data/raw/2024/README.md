# 2024 Raw Inputs

This folder stores the official-source raw inputs and overrides for the 2024 tournament dataset.

- `sources/rules.html` is the official Alliance Tournament XX rules post snapshot.
- `sources/sheet.html` is the linked official Google Sheet snapshot.
- `sources/static-values.gviz.json` is the structured Google Visualization response used for ship points.
- `source.json` is the normalized raw tournament source built from those artifacts.
- `overrides.json` is reserved for manual corrections if upstream source recovery is incomplete.
